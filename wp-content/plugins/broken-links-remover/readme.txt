== Description ==

Automatically remove the bad and broken links from  the view of human visitors and search engine robots.

**[Download now!](http://downloads.wordpress.org/plugin/broken-links-remover.zip)**

[FAQ](http://techblissonline.com/broken-links-remover/)

Some features:

* Broken Links Remover removes(hides) bad/broken links from the requestor's(search engine or human) view
* Bad/broken links dashboard

== Installation ==

1. Unzip into your `/wp-content/plugins/` directory. If you're uploading it make sure to upload
the top-level folder. Don't just upload all the php files and put them in `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit your  options (*Options - Broken Links Remover*) for Wordpress 2.3.x, (*Settings - Broken Links Remover*) for Wordpress 2.5.x

If you have to upgrade manually simply repeat the installation steps.
